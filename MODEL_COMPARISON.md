## Overview of Available Models

LLM Navigator now supports multiple AI models from different providers to give you the best possible analysis for your content. Each model has unique strengths and characteristics.

## OpenAI Models

### GPT-4
- **Best for**: Comprehensive analysis with strong pattern recognition
- **Strengths**: 
  - Excellent at understanding technical SEO elements
  - Strong semantic analysis capabilities
  - Detailed, actionable recommendations
- **Cost**: $0.03/1K input tokens, $0.06/1K output tokens
- **Available on**: Starter, Professional, Enterprise plans

## Anthropic Models

### Claude 3 Opus
- **Best for**: Deep reasoning and nuanced content understanding
- **Strengths**:
  - Superior reasoning capabilities
  - Excellent at understanding content context and relationships
  - Comprehensive analysis with nuanced insights
  - Longer context window for analyzing more content
- **Cost**: $0.015/1K input tokens, $0.075/1K output tokens
- **Available on**: Enterprise plan only

### Claude 3 Sonnet
- **Best for**: Balanced performance and cost
- **Strengths**:
  - Strong reasoning capabilities
  - Good balance of performance and cost
  - Effective for most content analysis needs
- **Cost**: $0.003/1K input tokens, $0.015/1K output tokens
- **Available on**: Professional and Enterprise plans

### Claude 3 Haiku
- **Best for**: Fast, cost-effective analysis
- **Strengths**:
  - Very fast response times
  - Most cost-effective Claude model
  - Good for basic content analysis
- **Cost**: $0.00025/1K input tokens, $0.00125/1K output tokens
- **Available on**: Starter, Professional, Enterprise plans

## Perplexity Models

### Perplexity Online
- **Best for**: Analysis with up-to-date information
- **Strengths**:
  - Access to recent web information
  - Excellent for analyzing content against current trends
  - Strong for competitive analysis with recent data
- **Cost**: $0.002/1K input tokens, $0.01/1K output tokens
- **Available on**: Professional and Enterprise plans

### Perplexity Offline
- **Best for**: Standard analysis without web search
- **Strengths**:
  - Fast analysis
  - Cost-effective
  - Good general knowledge base
- **Cost**: $0.001/1K input tokens, $0.005/1K output tokens
- **Available on**: Starter, Professional, Enterprise plans

## Model Selection Guide

### For SEO-Focused Analysis
- **Best choice**: GPT-4
- **Why**: Strongest understanding of technical SEO elements and structured data

### For Content Strategy
- **Best choice**: Claude 3 Opus or Sonnet
- **Why**: Superior understanding of content relationships and semantic connections

### For Competitive Analysis
- **Best choice**: Perplexity Online
- **Why**: Access to recent information about competitors and market trends

### For Cost-Effective Analysis
- **Best choice**: Claude 3 Haiku
- **Why**: Lowest cost per token while maintaining good analysis quality

## Best Practices

1. **Start with Claude 3 Haiku** for initial analyses to keep costs low
2. **Use GPT-4** for technical SEO-focused analyses
3. **Use Claude 3 Opus/Sonnet** for in-depth content strategy work
4. **Use Perplexity Online** when you need up-to-date competitive information
5. **Compare results** across different models for the most comprehensive insights
